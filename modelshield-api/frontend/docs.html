📖 API Documentation

Complete guide to integrate ModelShield API

🚀 Quick Start

Base URL:

http://localhost:8000

1. Create an API key from the API Keys page.

2. Include it in your requests as a Bearer token.

💬 Chat Completions

POST /v1/chat/completions

Request Headers

Authorization: Bearer ms_your_api_key_here
Content-Type: application/json

Request Body

{
  "model": "llama-3.1-8b-instant",
  "messages": [
    { "role": "system", "content": "You are a helpful assistant." },
    { "role": "user", "content": "Explain gravity in simple terms" }
  ],
  "temperature": 0.7,
  "max_tokens": 1024
}

Example using cURL

curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Authorization: Bearer ms_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama-3.1-8b-instant",
    "messages": [{"role":"user","content":"Hello!"}]
  }'

Python Example

import requests

response = requests.post(
    "http://localhost:8000/v1/chat/completions",
    headers={"Authorization": "Bearer ms_your_key"},
    json={
        "model": "llama-3.1-8b-instant",
        "messages": [{"role": "user", "content": "Hello!"}]
    }
)
print(response.json()["choices"][0]["message"]["content"])
